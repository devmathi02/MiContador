## 📌 ¿Qué es miContador?

miContador es una aplicación de consola para gestionar tus finanzas personales. Te permite:

- ✅ Registrar **ingresos** (salario, freelance, inversiones...)
- ✅ Registrar **gastos** (alimentación, transporte, vivienda...)
- ✅ Ver tu **historial completo** de transacciones
- ✅ Consultar tu **saldo actual** en tiempo real
- ✅ Clasificar movimientos por **categorías** con emojis
